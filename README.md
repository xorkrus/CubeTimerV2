Таймер-куб на RP2040
======
 * Устройство представляет из себя куб, размером 6x6x6 сантиметров.
 * С одной из сторон расположен круглый экран GC9A01 диагональю 1.28" иразрешением 240*240 точек.
 * С другой стороны расположен разъем зарядки/интерфейс найстройки.
 * Имеется кнопка для включения.

 * Путем установки куба на одну из шести граней, запускается один из 6 таймеров с визуализацией отсчета на экране. По окончании отсчета раздрается звуковой сигнал.

# Команды управления умным кубом (версия прошивки 7.4)

Все команды отправляются через UART (Serial) и завершаются символом новой строки (`\n`).

## Общие команды

| Команда | Описание |
|---------|----------|
| `help` | Показать эту справку |
| `ver` | Вывести версию прошивки и дату компиляции |
| `reboot` | Перезагрузить устройство |
| `save` | Сохранить текущие настройки в EEPROM |
| `list` | Показать все текущие настройки (шрифты, стороны, таймеры, цвета и т.д.) |
| `list <n>` | Показать детальную информацию о шрифте с номером `n` (0–9) |

---

## Режимы мониторинга

| Команда | Описание |
|---------|----------|
| `sens` | Войти в режим вывода данных акселерометра (раз в секунду). Для выхода отправить любой символ. |
| `thresh` | Войти в режим отладки порогов: показывает текущие пороги и измерения акселерометра. Выход – любой символ. |

---

## Команда `set` – настройка параметров

Общий синтаксис:  
`set <категория> <подкоманда> <параметры>`

### 1. Категория `timer` – настройки таймера для каждой стороны

| Подкоманда | Параметры | Описание |
|------------|-----------|----------|
| `sfs` | `<side> <fontIdx>` | Выбрать шрифт для стороны (side 1..6, fontIdx 0..9) |
| `fc` | `<fontIdx> <x>:<y>` | Установить **базовую** коррекцию положения шрифта (в пикселях) |
| `fcs` | `<side> <fontIdx> <x>:<y>` | Установить **дополнительную** коррекцию для конкретной стороны и шрифта |
| `fs` | `<fontIdx> <value\|+d\|-d>` | Установить масштаб шрифта (1.0 = 100%). Можно указать абсолютное значение, либо относительное (`+0.2`, `-0.1`) |
| `tm` | `<side> <seconds>` | Установить время таймера для стороны (в секундах) |
| `sc f` | `<side> <RRGGBB>` | Установить цвет текста (шрифта) для стороны (шестнадцатеричный, например `FFFFFF` для белого) |
| `sc b` | `<side> <RRGGBB>` | Установить цвет фона для стороны |

**Примеры:**  
| Команда | Описание |
|---------|----------|
| **set timer sfs 3 5** | стороне 3 назначить шрифт 5 |
| **set timer fc 2 10:-5** | базовая коррекция шрифта 2: +10 по X, -5 по Y |
| **set timer fcs 3 2 2:0** | дополнительно для стороны 3 шрифта 2: +2 по X, 0 по Y |
| **set timer fs 4 +0.2** | увеличить масштаб шрифта 4 на 20% |
| **set timer tm 5 90** | сторона 5 будет таймер 90 секунд (1:30) |
| **set timer sc f 1 FF0000** | красный текст на стороне 1 |
| **set timer sc b 1 000000** | чёрный фон на стороне 1 |

---

### 2. Категория `bat` – настройки отображения заряда батареи

| Подкоманда | Параметры | Описание |
|------------|-----------|----------|
| `fcs` | `<side> <x>:<y>` | Коррекция положения процентов батареи на конкретной стороне |
| `fs` | `<value\|+d\|-d>` | Масштаб шрифта батареи |
| `sc f` | `<side> <RRGGBB>` | Цвет текста батареи для стороны |
| `font` | `<fontIdx>` | Выбрать шрифт для батареи (один на все стороны) |

*Примечание:* для батареи базовая коррекция (`fc`) не используется – только дополнительная на сторонах.

**Примеры:**  
| Команда | Описание |
|---------|----------|
| **set bat fcs 2 5:-3** | батарея на стороне 2: +5 по X, -3 по Y |
| **set bat fs 1.2** | масштаб шрифта батареи 1.2 |
| **set bat sc f 3 00FF00** | зелёный текст батареи на стороне 3 |
| **set bat font 1** | использовать шрифт 1 для батареи |

---

### 3. Категория `message` – настройки текстовых сообщений (WAITING, OFF и т.п.)

| Подкоманда | Параметры | Описание |
|------------|-----------|----------|
| `fc` | `<x>:<y>` | Коррекция положения сообщения (одна на все стороны) |
| `fs` | `<value\|+d\|-d>` | Масштаб шрифта сообщений |
| `sc f` | `<RRGGBB>` | Цвет текста сообщения |
| `sc b` | `<RRGGBB>` | Цвет фона сообщения |
| `font` | `<fontIdx>` | Выбрать шрифт для сообщений |

**Примеры:**  
| Команда | Описание |
|---------|----------|
| **set message fc -5:10** | сместить сообщение: -5 по X, +10 по Y |
| **set message sc f FFFFFF** | белый текст |
| **set message sc b 000000** | чёрный фон |
| **set message font 0** | использовать шрифт 0 для сообщений |

### 4. Категория `sn` – настройка чувствительности (сенсор)

| Подкоманда | Параметры | Описание |
|------------|-----------|----------|
| `sn f` | `<value>` | Порог стабильности для фиксации стороны (в м/с²). По умолчанию 0.4 |
| `sn t` | `<value>` | Порог обнаружения тапа (в м/с²). По умолчанию ~24.5 (2.5g) |

**Примеры:**  
| Команда | Описание |
|---------|----------|
| **set sn f 0.3** | более чувствительная фиксация стороны |
| **set sn t 30.0** | повысить порог тапа |

### 5. Категория `global` – общие параметры

| Подкоманда | Параметры | Описание |
|------------|-----------|----------|
| `global confirm` | `<sec>` | Длительность этапа подтверждения (сек). По умолчанию 3 |
| `global shutdown` | `<sec>` | Время показа надписи OFF перед выключением (сек). По умолчанию 10 |
| `global led` | `<0-255>` | Яркость встроенного светодиода |

**Примеры:**  
| Команда | Описание |
|---------|----------|
| **set global confirm 5** | подтверждение 5 секунд |
| **set global shutdown 15** | OFF показывать 15 секунд |
| **set global led 100** | яркость светодиода 100 |

## Примечания

- Все цвета задаются в шестнадцатеричном формате RRGGBB (например, `FF8000` – оранжевый).
- Координаты `x` и `y` указываются в пикселях. Положительные значения сдвигают вправо/вниз, отрицательные – влево/вверх.
- Масштаб шрифта – множитель относительно исходного размера шрифта. Можно указывать как десятичную дробь (например `1.5`).
- После изменения параметров необходимо выполнить `save` для сохранения в EEPROM. После перезагрузки (`reboot`) настройки применятся.
- Нумерация сторон: 1–6 (соответствуют граням куба).
- Нумерация шрифтов: 0–9 (файлы mf00.h … mf09.h).

Удачной настройки!

## Компоненты

1. **Микроконтроллер**: Waveshare RP2040-Zero  
   - основан на Raspberry Pi RP2040  
   - встроенный NeoPixel на GP16  

2. **Дисплей**: GC9A01A 240×240 SPI  
   - цветной круглый IPS-дисплей 1.28" 

3. **Акселерометр/гироскоп**: LSM6DS3 (или LSM6DS3TR)  
   - интерфейс I2C  

4. **Адресный светодиод**: NeoPixel (встроен на плате RP2040-Zero)  

5. **Пьезодинамик**: пассивный зуммер (подключается к GP27)  

6. **Управление питанием**:  
   - транзисторный ключ на GP15 (HIGH – удерживать питание, LOW – отключить)  
   - кнопка включения (для подачи питания, а софт может отключить себя сам)  

7. **Измерение напряжения батареи**:  
   - делитель 1:1 (два резистора по 100 или 200 кОм) постоянно подключён к батарее  
   - средняя точка делителя – на GP26 (ADC0)

## Схема делителя батареи

Батарея (литий-полимер, 3.0–4.2 В) подключена к делителю:
```
Bat+ ──┬── R1 (100 кОм) ──┬── ADC (GP26)
       │                  │
       └── R2 (100 кОм) ──┴── GND
```
- Коэффициент деления: (R1+R2)/R2 = 2.  
- Напряжение на ADC = Vbat / 2.  
- При Vbat = 4.2 В на ADC ≈ 2.1 В (безопасно для 3.3 В логики).  
- Опорное напряжение АЦП – 3.3 В.

---

## Управление питанием

- При подаче внешнего питания (через кнопку или USB) включается RP2040.
- В `setup()` пин GP15 переводится в HIGH, что удерживает транзистор открытым – питание остаётся даже после отпускания кнопки.
- Для выключения (по таймеру или команде) программа переводит GP15 в LOW, транзистор закрывается – питание отключается.

---

## Примечания

- Дисплей GC9A01A работает по SPI, библиотека TFT_eSPI должна быть настроена на RP2040 с соответствующими пинами.
- I2C подтяжки (2.2–10 кОм) должны быть установлены на линии SDA/SCL (обычно есть на модуле LSM6DS3, если нет – добавить внешние).
- Пьезодинамик подключается напрямую, без дополнительных резисторов (ток ограничивается самим выводом RP2040).
- Все свободные пины можно использовать для расширения.
